# Realty App

## Умная система управления недвижимостью и арендой

Современное full-stack приложение для агентств недвижимости и частных арендодателей. Автоматизирует учёт объектов, контроль договоров и взаимодействие с клиентами.

> **Стек:** FastAPI (async) · PostgreSQL · SQLAlchemy 2.0 · Vue 3 · TypeScript · Vite · Alembic

![Интерфейс приложения](documentation/photo_2026-06-13%2015.06.14.jpeg)

## Возможности

**Объекты недвижимости** — создание, редактирование, удаление, просмотр списка объектов с фильтрацией и привязкой к владельцу.

**Договоры аренды** — создание договора через поиск арендатора по email, автоматическое формирование графика платежей на весь срок аренды, статусы (ожидает подтверждения / активен / завершён).

**Платежи** — автоматическое создание платежей по договору, отправка подтверждения оплаты арендатором, подтверждение/отклонение владельцем, отслеживание задолженности.

**Заявки** — создание обращений арендаторами, изменение статусов владельцем (новая / в работе / выполнена).

**Статистика** — общий и ожидаемый доход за месяц, количество объектов и активных договоров, график доходов.

**Единая модель пользователя** — один аккаунт для ролей владельца и арендатора, переключение режимов без повторной авторизации.
